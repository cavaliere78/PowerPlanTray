# PowerPlanTray
Script Windows PowerShell 5.1 per la visualizzazione e la modifica del profilo energetico tramite icona nella tray icon in basso a destra su Windows.

Lo script va lanciato con il seguente comando:
powershell.exe -sta -WindowStyle Hidden -ExecutionPolicy Bypass -File ".\PowerPlanTray.ps1"
